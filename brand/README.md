# Geekbyte Brand Assets

This folder contains all brand materials, logos, and design guidelines for Geekbyte.

## Folder Structure

### `/presentations/`
Place your PowerPoint presentations and pitch decks here.
- Brand presentations
- Company overview decks
- Marketing materials

### `/logos/`
All logo files and variations.
- Full logos (color, white, black)
- Icon/symbol only versions
- Different file formats (SVG, PNG, etc.)
- Logo usage guidelines

### `/assets/`
Other brand materials and resources.
- Brand color swatches
- Typography files
- Photography guidelines
- Icon sets
- Pattern libraries
- Any other brand assets

## How to Use

When working with the ui-designer agent, reference these materials:

```
"ui-designer, create a [feature] following our brand guidelines.
Brand assets are in /brand/ folder including:
- Presentations: /brand/presentations/
- Logos: /brand/logos/
- Guidelines: /brand/assets/"
```

## Instructions for Adding Files

1. Copy your PowerPoint presentations to `/brand/presentations/`
2. Copy all logo files to `/brand/logos/`
3. Copy any other brand materials to `/brand/assets/`

Once files are added, Claude can read and analyze them to understand your branding.
